# Features Documentation

## Current Features

### Technical Indicators
- **Moving Average (MA)**: Calculates the average price over a specific period.
- **Relative Strength Index (RSI)**: Measures the speed and change of price movements.
- **Bollinger Bands**: A volatility indicator that consists of a middle band and two outer bands.

### Signals
- **Buy Signal**: Generated when the price crosses above a specific indicator threshold.
- **Sell Signal**: Generated when the price crosses below a specific indicator threshold.

### UI Components
- **Dashboard**: Displays key performance indicators and charts.
- **Signal Alerts**: Notifies users of buy/sell signals via the UI and email.
- **Customizable Charts**: Allows users to modify chart parameters (timeframe, indicators).

## Planned Features

### Upcoming Technical Indicators
- **MACD**: Moving Average Convergence Divergence, a trend-following momentum indicator.
- **Fibonacci Retracement**: Technical analysis tool to identify potential reversal levels.

### Future Signals
- **Divergence Signals**: Identifying divergence between price action and indicators for signals.

### Future UI Enhancements
- **Mobile Responsiveness**: Ensuring the UI is fully functional on mobile devices.
- **User Personalization Options**: Allow users to customize themes and layouts.
